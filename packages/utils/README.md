# @orbital/utils

This package will contain a few utilities. Currently:

1) Logger. A terse standalone logger. Emits single-line logs with `HH:MM:SS.mmm` timestamps, caller source locations for warn and error levels, and automatic caller stack-trace extraction for troubleshooting. The logger captures console events and formats multiline output inline to keep logs scannable and compact.
